# Intrusion-Detection-Based-on-OpenCV 基于OpenCV的入侵检测
## 项目简介
1. 通过摄像机连续观测某一动态场景；

2. 获取摄像机实时视频流数据，当检测到物体入侵时以一定时间间隔保存图片至本地；

3. 对保存图像进行均值/中值滤波；

4. 采用形态学处理检测入侵物体，并显示入侵物体图像；

5. 提取物体图像的灰阶共生矩阵，同时实现在该物体图像上的灰阶分割与图像细化；

6. 显示物体在图像上的移动轨迹；

7. 预测物体的移动速度，精度不低于60%。
## 运行方式
运行main.cpp即可
## 使用环境
OpenCV 2.4.13.6
## 语言
C++
